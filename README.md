**cd** - Debe de ser usado con una carpeta y directorios que tengas en tu maquina.

![](/imagenes/cd.png)

**pwd** - Al usarse en la terminal te mostrara el lugar donde estas situado.

![](/imagenes/pwd.png)

**mkdir** - Debe de usarse con un nombre seguido del comando, creara una carpeta con dicho 
nombre.

![](/imagenes/mkdir.jpeg)

**nano** - Al igual que mkdir debe de ponerse un nombre seguido del comando, solo que este  deberá terminar en .txt o cualquier otro formato, ya que creara un documento con dicho nombre introducido para editarlo.

![](/imagenes/nano.png)

## Variaciones en los comandos básicos

**ls -la** 
Funciona igual que un ls normal pero con la diferencia que puedes ver los archivos y carpetas a detalle y las cosas que están ocultas, l es para detallado y la a es para lo oculto.
También puedes usarlo por separado "ls -a" "ls -l" y cumplirá la función dependiendo de la letra que coloques.

![](/imagenes/lsla.png)

**cd ..**
sirve para retroceder entre carpetas, archivos o directorios, si quisieras retroceder dos o mas directorios al mismo tiempo, tendrías que separar los dos puntos con un slash, por cada dos puntos retrocederás un directorio "cd ../.." .

![](/imagenes/cd2.png)

**rm -r**
Tiene mas permisos para eliminar cosas, con el rm normal no se pueden borrar carpetas u otras cosas, con este si podrás.

![](/imagenes/-r.png)
